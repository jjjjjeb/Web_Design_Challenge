Comparisons

The purpose of this project was to analyse how weather changes as you get close to the equator. To accomplish this analysis we first pulled data from the OpenWeatherMap API to assemble a dataset on over 500 cities.

After assembling the dataset, we used Matplotlib to plot various aspects of the weather vs latitude. Factors we included were temperature, cloudiness, windspeed, and humidity, as you can see in the visualizations above.

To see a sample of the data set pulled from OpenWeatherMap check out the data page.
